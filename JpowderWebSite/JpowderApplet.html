JpowderApplet

JpowderApplet uses the same base code as the Jpowder application, but displays just one chart. The purpose of JpowderApplet is to allow the display of data, not just as static images, but in such a way, that viewers of your web pages can interact visually with the data. Such plots may for example be used in E-learning.
