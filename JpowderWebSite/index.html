Lightweight powder diffraction data visualizer

  • Optimised for visual inspection of powder diffraction data
  • Rapid access to the software from any location using Java Web Start technology
  • Runs on Window, Mac and Linux
  • Availability as plug-in to allow use in other applications
  • Reads powder data format including CIF(*.cif), XYE, XY and ISIS-GSAS(*.gss)
  • Open Source, free and accessible to any user
  • NEWS: Jpowder paper recently published
  • For more info see also menu links above including About and the User-Wiki
