Problems installing Jpowder?

To run Jpowder you need to install JRE version 1.6 or later.

For general instruction to install Java on any PC see the Sun (now Oracle) Java web site http://www.java.com/download.

Stopped support of Java Web Start download

We have stopped the support of Java Web Start for downloading Jpowder since this technology have been found not to be supported well on different platforms. Therefore to install Jpowder download the file: Jpowder.jar. If you have any Java JRE installed it should run.
