In general

You need to have Java installed and by default Java does not appear to be installed on the Ubuntu systems we have tested.

The simplest is to open Firefox and go to http://www.java.com/en/download/installed.jsp followed by pressing the big red button. Then if Java isn't installed Firefox will ask you to install the missing plugins at the top of the browser. Hit the "install missing plugins" button and say yes to install Icedtea.

In general there are two options for installing JRE+plugin: the Sun/Oracle version and the OpenJDK/IcedTea version.

The Ubuntu web-site includes this page: help.ubuntu.com/community/Java.

Ubuntu 10.04 tests

Java is not installed by default, or at least not the IcedTea plugin. To install OpenJDK/IcedTea simply follow the instructions above. To install the Sun/Oracle version use the package manage and select the packages: sun-java6-jre and sun-java6-plugin.
