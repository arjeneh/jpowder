Information about Release 1

Its scope is: 2D visualization of powder data + simple tools for manipulating these.

A paper has been published in connection with the completion of release 1, see here, please cite this paper in any work where Jpowder is used. 

Future Release 2

During testing of Jpowder Release 1, the most frequent future request was: could Jpowder be extended to allow 3D visualisation of powder data. This means potentially read hundreds (or thousands!) of powder datasets and plotting these within one “3D” figure. For instance, where these datasets are measured at slightly different temperatures and the “3rd“ dimension of the “3D” figure is temperature.

Work is in progress for adding this feature (see for a proposed GUI design and the SVN branch named Jpowder3D).

Tentative release date for this release is Marsh 2011.

 

This project is Open Source and Criticism, Comment and Contributions are welcome an any time.

Point of contact

Bugs & suggestions for new feature: In the first instance please consider submitting an issue at
http://code.google.com/p/jpowder/issues/list by clicking the ‘New issue’ link on that web page.
When filling in a ‘New issue’ don’t worry about filling in any of the boxes except for the
‘Summary’ and ‘Description’ box, we will do the rest.

You like to discuss something? Well, why don’t you try: http://groups.google.com/group/jpowder  

Add comments to a blog! We have a Jpowder blog, although to be honest we are not sure how useful
this blog will be, and it may be removed in the future. But meanwhile you are more than welcome
to prove the opposite!

Still having questions or fancy contributing to Jpowder? Try emailing A Markvardsen or M Arjeneh and we will reply as quickly as we can.

 

Jpowder uses the following software libraries: JFreeChart, iText and JXLayer.
