In general

Two flavours of Java is evailable on Linux: the Sun/Oracle and OpenJDK/IcedTea versions of JRE+plugin. Fedora and Redhat by default have OpenJDK/IcedTea installed and our experience is that this works fine.

The Fedora web-site includes this page: https://fedoraproject.org/wiki/Java.

Fedora Core 10-11 and 13 tests

Jpowder works from a standard install. If for some reason OpenJDK/IcedTea has been uninstalled then either install it from the package manager (java-1.6.0-openjdk and java-1.6.0-openjdk-plugin) or simply by opening http://www.java.com/en/download/installed.jsp in Firefox followed by pressing the big red button. Then if Java isn't installed Firefox will ask you to install the missing plugins at the top of the browser. Hit the "install missing plugins" button and say yes to install Icedtea.
