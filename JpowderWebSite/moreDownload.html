Problems installing Jpowder?

To run Jpowder you need to install JRE version 1.6 or later, and the box below shows you what version of JRE your have installed (if any):

For general instruction to install Java on any PC see the Sun (now Oracle) Java web site http://www.java.com/download.

Installing a JRE is in particular trivial on Windows, and some Linux distributions it is preinstalled (no experience with Macs). We have put some additional install together for a number of Linux distributions, see: Ubuntu, Fedora/Redhat.

Working on a restricted PC?

Try downloading this file: Jpowder.jar directly. If you have any Java JRE installed it should run. The main disadvantage of installing Jpowder this way is that you will not receive automatic Jpowder updates.
